# Cenciel AI Brand-Ops Pipeline OS
> 뷰티 브랜드 매니저를 위한 3단계 자동화 AI 마케팅 운영체제 💄✨

### 현재는 mvp 정도로 전체적인 기능은 구현 x

`Cenciel AI Brand-Ops`는 화장품 단 한 번의 단어/URL 입력 또는 리뷰 파일 업로드만으로 **시장 레이더 탐지부터 소비자 VOC 분석, 그리고 멀티채널(인스타그램 카드뉴스, 네이버 블로그 스마트에디터, 숏폼 릴스/쇼츠 대본) 맞춤형 카피라이팅**을 10초 만에 완결하는 고기능성 마케팅 파이프라인 MVP 시스템입니다.

현업 뷰티 브랜드 매니저들이 기획서 작성 및 카피 추출에 겪는 막대한 대기 시간과 공수를 극적인 속도로 대폭 축소합니다.

---

## 🚀 주요 핵심 기능 (Key Features)

### 📊 1단계. 시장 경쟁 레이더 (Market Radar)
* **제품 정보 스크래핑**: 스마트스토어, 올리브영 등 경쟁 제품 URL이나 뷰티 키워드를 간편 크롤링합니다.
* **지능적 트렌드 추적**: 크롤러 데이터를 토대로 소셜 미디어/쇼핑몰 내 버즈 가중치를 파악하여 트렌드 키워드 Top 5를 다이내믹 렌더링합니다.
* **차별화 상위 포지셔닝 제안**: 경쟁 제품의 핵심 독자 성분/강점(USP) 요약 및 우리 브랜드가 파고들 타겟 대상 공략 전략을 자동으로 도출합니다.

### 🧪 2단계. 고객 VOC 인사이트 분류 (VOC Insights)
* **리뷰 파일 업로드 (Excel / CSV)**: 엑셀 데이터를 업로드하면 지능형 파서가 특수 열(별점, 텍스트)을 오토 시퀀싱(Auto-detection)합니다. 
* **정성 텍스트 복사-붙여넣기**: 소셜/자사몰에서 긁어온 정성 의견 텍스트를 줄바꿈하여 붙여놓고 바로 즉석 텍스트 마이닝을 할 수 있습니다.
* **Pain Point 및 최상위 아이디어 처방**: 소비자의 강한 불만 사항(Pain Points)과 평점 통계를 분류해주며, 해당 문제를 해결할 혁신적인 상품 개발/개선 방향성 제안 테이블을 즉시 도출합니다.

### ✍️ 3단계. 멀티 채널 포스팅 빌더 (Multi-Channel Copywriter)
* **원클릭 자동 연계**: 2단계에서 도출된 핵심 소구점(Selling Point) 카드를 클릭하면 바로 3단계 빌더 텍스트 상자로 자동 복사 후 화면이 자동 스위칭됩니다.
* **인스타그램 미러 피드 + 카드뉴스 스크립트**: 5페이지 분량의 이탈 방지 슬라이드 기획물 가이드와 이모지가 풍부히 내장된 캡션 및 추천 태그를 빌딩하며, 스마트폰 시뮬레이터 목업의 도트 개수까지 연계해 현장감 있게 미리 볼 수 있습니다.
* **네이버 공식 블로그 내용 구성**: 스마트에디터 배포에 최적화된 긴 호흡의 제목/본문/결론 레이아웃을 SEO 친화적으로 제공합니다.
* **숏폼 영상 스크립트 구성**: 15초 러닝타임에 맞춘 비디오 화각 설명(Visual)과 립스/릴스 전용 나레이션 오디오 대본(Audio) 규격을 시퀀스로 지원합니다.
* **원클릭 복사 시스템**: 작업한 모든 원고는 우측 상단의 클립보드 복사 버튼으로 즉석 전달이 가능합니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Backend / API**: FastAPI, Uvicorn (ASGI)
* **Data Processing**: Pandas, OpenPyxl (Excel/CSV 자동 전처리)
* **Web Crawler**: BeautifulSoup4, HTTPX
* **LLM Engine**: OpenAI API (`gpt-4o-mini` 가성비 모드 / `gpt-4o` 퀄리티 모드 하이브리드 탑재)
* **Frontend**: HTML5, Vanilla CSS3 (Custom Dark-mode Glassmorphism), Vanilla JS (ECMAScript 6)

## ⚙️ 시작 및 가동 가이드 (Setup & Runs)

### 1. 가상환경 세팅 및 라이브러리 설치
```bash
# 가상환경 구성
python -m venv .venv

# 가상환경 진입 (Windows)
.venv\Scripts\activate
# (Mac/Linux의 경우: source .venv/bin/activate)

# 패키지 의존성 다운로드
pip install -r requirements.txt
```

### 2. 환경 설정 파일 (`.env`) 설정
프로젝트 최상위 디렉토리에 **`.env`** 파일을 생성하고 다음과 같이 API 키와 사용할 모델을 입력합니다. (키가 주입되지 않을 경우 오프라인 뷰티 도메인 캐싱 프리셋으로 작동합니다.)

```env
# OpenAI API Key 실물 (필수)
OPENAI_API_KEY=sk-your-openai-api-key-here

# AI 모델 모드 (선택사항, 생략 시 gpt-4o-mini)
# - gpt-4o-mini: 속도 최우선 + 초저가
# - gpt-4o: 가장 자연스럽고 유려한 고급 마케팅 작문
OPENAI_MODEL=gpt-4o-mini
```

### 3. 로컬 서버 시작
```bash
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```
가동 후 웹 브라우저에서 **`http://127.0.0.1:8000`** 주소로 접속해 즉시 사용을 확인하실 수 있습니다.

### 4. 로컬 통합 테스트 실행
서버 안정성과 전 Endpoint 연동 무결성을 스크립트로 신속하게 자가 정진할 수 있습니다.
```bash
python test_endpoints.py
```
